Analista de Dados

Um analista de dados possui um conjunto de técnicas que auxiliam na tomada de decisões estratégicas dentro dos mais diversos segmentos empresariais. Ex: Criação de KPI's, relatórios, dashboards web e mobile com Excel, Power BI e Python.
